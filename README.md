# Competentieroos

Competentieroos is een eenvoudige website voor Scouting Nederland waarmee leidinggevenden zichzelf kunnen beoordelen op belangrijke competenties. De site helpt gebruikers om een korte zelfreflectie te maken, te zien waar ze sterk in zijn en waar nog ruimte is voor ontwikkeling.

De website is opgebouwd rond twee routes:
- BWS voor Bevers, Welpen en Scouts
- ER voor Explorers en Roverscouts

Gebruikers beantwoorden een reeks vragen, krijgen direct feedback op hun scores en kunnen hun resultaten bekijken in een overzichtelijke resultatenpagina.

## Features
- Kies tussen BWS en ER
- Beantwoord 27 competenties
- Bekijk directe resultaten in een overzichtelijke resultatenpagina
- Sla je voortgang op in de browser
